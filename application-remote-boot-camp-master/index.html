What is Front End?

The front end of the website is the part that the user interacts with. It presents the possibilities of the website in the same way a cars instrument panel, steering wheel and gear lever makes it possible for the driver to interact with and operate the car. A front-end developer creates an interface using code such as HTML, CSS and Javascript.
The goal of designing a site is that the user easily can find the relevant information and perform their tasks, the design also needs to work in different devices, browsers and operating systems etc.
